# 🏗️ 系统架构

## 整体设计

Hypnos Dream System 是一个**夜间认知循环系统**，在用户入睡后自动执行，通过五个阶段完成信息整理、知识内化、创意联结和自我进化。

## 核心组件

### 1. 触发器（Trigger）

```
┌─────────────────────────────────────────┐
│  触发条件                                │
│  ├─ 用户说"晚安" → 立即触发              │
│  ├─ 30分钟无交互 → 延迟触发              │
│  └─ Cron 定时（凌晨 2:00）→ 保底触发     │
└─────────────────────────────────────────┘
```

### 2. 数据收集器（Collector）

```
┌─────────────────────────────────────────┐
│  数据源                                  │
│  ├─ session_search → 今日对话记录        │
│  ├─ memory → 用户画像 + 已知偏好         │
│  ├─ skills → 技能使用情况                │
│  └─ cron logs → 定时任务执行记录         │
└─────────────────────────────────────────┘
```

### 3. 五阶段处理管道

```
数据收集
    │
    ▼
┌─────────────┐
│ Mnemosyne   │ → 事实性总结（已完成/未完成/新知/待确认）
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Epimetheus  │ → 知识三元组 + 核心记忆 + 用户画像更新
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Prometheus  │ → 创意联结（跨主题灵感）
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Evolution   │ → 行为补丁（明天怎么做得更好）
└──────┬──────┘
       │
       ▼
┌─────────────┐
│ Dream Log   │ → 结构化日志（Markdown）
└─────────────┘
```

### 4. 存储与推送

```
梦境日志
    │
    ├─ 本地存储 → ~/.hermes/dreams/YYYY-MM-DD.md
    │
    └─ 次日推送 → Lark/飞书（早 8:00）
```

## 数据流

```
今日对话 ──→ session_search ──→ Mnemosyne ──→ Epimetheus
                │                    │              │
                │                    │              ▼
                │                    │         知识三元组
                │                    │         核心记忆
                │                    │         用户画像
                │                    │              │
                │                    ▼              │
                │              浅睡总结 ─────────────┤
                │                                   │
                ▼                                   ▼
           Prometheus ─────────────────────── Evolution
                │                                   │
                ▼                                   ▼
           灵感泡沫                             进化补丁
                │                                   │
                └──────────────┬────────────────────┘
                               │
                               ▼
                        Dream Log（梦境日志）
                               │
                               ▼
                        本地存储 + 次日推送
```

## 技术选型

| 组件 | 选择 | 理由 |
|:----:|:----:|:----:|
| 调度 | Hermes Cron | 原生支持，无需外部依赖 |
| 数据收集 | session_search | FTS5 全文检索，快速 |
| 知识存储 | Memory + Skills | 持久化，跨会话 |
| 日志格式 | Markdown | 可读性好，Lark 原生支持 |
| 推送渠道 | Lark/飞书 | 用户主要通讯平台 |
